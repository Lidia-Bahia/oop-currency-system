# OOP Currency System (Java)

Projeto desenvolvido para prática de Programação Orientada a Objetos em Java.

## 💡 Descrição
O sistema simula um cofrinho digital que permite armazenar diferentes tipos de moedas e realizar operações básicas como adição, remoção, listagem e conversão de valores.

## ⚙️ Funcionalidades
- Adicionar moedas (Real, Dólar e Euro)
- Remover moedas específicas
- Listar moedas armazenadas
- Calcular o total convertido em Real

## 🧠 Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Herança
- Polimorfismo
- Encapsulamento
- Uso de ArrayList
- Entrada de dados com Scanner

## ▶️ Como executar
1. Compile os arquivos Java
2. Execute a classe `Principal`
3. Interaja com o menu no console

## 📌 Observações
Este projeto tem fins educacionais e foi desenvolvido como parte de atividades acadêmicas.
